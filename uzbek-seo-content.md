## 7. TECHNICAL SEO TAKLIFLARI

### Mobile-First Optimization
- Mobile ko'rinish birinchi loyihalash
- Touch-friendly buttonlar (min 48px)
- Readable matn (min 16px)
- No horizontal scroll

### Page Speed
- Rasmlar: WebP format, lazy loading
- CSS/JS: Minify, defer
- Font: display swap, preconnect
- Target: LCP < 2.5s

### Image Compression
- WebP yoki AVIF format
- Explicit width/height
- Loading="lazy"

### Heading Structure
- Faqat bitta H1
- H2-H6 mantiqiy ketma-ketlikda
- Har bir heading semantik bog'liq

### Crawlability
- Sahifa uzunligi: < 3000 so'z
- Anchor navigatsiya mavjud
- Noindex yo'q

### Indexability
- `<meta name="index, follow">`
- Canonical URL belgilangan
- Til belgilangan: lang="uz"

### Internal Anchor Navigation
- ID'lar aniq (#xizmatlar)
- Smooth scroll js orqali

### Meta Tags
- Title: "Sayt Yaratish | Premium Web Dizayn"
- Description: Zamonaviy sayt...

### Open Graph
```html
<meta property="og:title" content="Sayt Yaratish | Premium Web Dizayn">
<meta property="og:description" content="Zamonaviy va sifatli sayt yaratish xizmati...">
<meta property="og:type" content="website">
```
